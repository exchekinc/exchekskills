## Input-hash computation (canonical)

```
1. Build inputs object: all user-provided fields, with
   keys sorted alphabetically, string values trimmed,
   empty strings omitted, dates normalized to ISO 8601.
2. JSON-serialize: JSON.stringify(inputs) with no extra whitespace.
3. Hash: SHA-256 of the UTF-8 byte sequence.
4. Truncate: first 12 hex characters of the lowercase digest.
```

This makes the hash stable across re-runs with the same inputs and auditable.
